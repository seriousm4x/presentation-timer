# Presentation Timer

A simple, distraction-free countdown timer for presentations, talks, and meetings.

## Features

- Set any countdown duration (up to 24 hours)
- Quick time presets for common durations
- Pause and resume the timer at any time
- Background color changes as time runs out
  - 60% elapsed → yellow
  - 80% elapsed → red
- Counts up after reaching zero to show overtime
- Responsive layout that adapts to any screen size
- Progress bar at the top for a clear visual cue

---

Perfect for presenters who want a clean and reliable timer.
